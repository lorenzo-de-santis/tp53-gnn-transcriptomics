# TP53 Classification with Graph Neural Networks

This research project investigates the use of graph neural networks for **TP53 mutation-status classification from single-cell transcriptomic data**.

The study focuses on rigorous, leakage-controlled evaluation across cell lines and datasets, with particular attention to biological signal, dataset shift, and model generalization. The project is currently being developed into a scientific manuscript with **Prof. Andrea Tangherloni, Bocconi University**.

## Research status

The manuscript is currently in preparation. To protect unpublished methods and results, the source code, experimental configurations, and detailed analyses are temporarily kept private.
